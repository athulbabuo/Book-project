# 📚 Book Management System (Django Project)

## 🔹 Overview

This is a web-based application developed using Django that allows users to manage books efficiently.
The application supports basic CRUD (Create, Read, Update, Delete) operations with a simple and user-friendly interface.

---

## 🚀 Features

* ➕ Add new books
* 📖 View book list
* ✏️ Update book details
* ❌ Delete books
* 🌐 Simple and clean UI

---

## 🛠️ Technologies Used

* Python
* Django
* HTML
* CSS
* SQLite (default Django database)

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/athulbabuo/bookproject.git
```

2. Navigate to the project folder

```
cd bookproject
```

3. Create a virtual environment

```
python -m venv venv
```

4. Activate the virtual environment

* Windows:

```
venv\Scripts\activate
```

* Mac/Linux:

```
source venv/bin/activate
```

5. Install dependencies

```
pip install -r requirements.txt
```

6. Run migrations

```
python manage.py migrate
```

7. Start the development server

```
python manage.py runserver
```

8. Open in browser

```
http://127.0.0.1:8000/
```
